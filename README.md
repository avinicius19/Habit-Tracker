Habit Tracker 

Um projeto simples e funcional de rastreador de hábitos, desenvolvido com React, focado em treinar lógica, estado, manipulação de arrays e interação do usuário.

O objetivo do projeto é permitir que o usuário crie hábitos, marque os dias da semana em que realizou cada hábito e ganhe pontos de motivação conforme a consistência.


Funcionalidades:

- Adicionar novos hábitos

- Marcar e desmarcar dias da semana para cada hábito

- Excluir hábitos individualmente

- Sistema de pontos (ganha pontos ao marcar dias)

- Estilização simples e intuitiva

- Uso de estados e lógica 100% em React

Tecnologias Utilizadas:

- React

- JavaScript (ES6+)

- CSS3

- React Icons

Conceitos Trabalhados:

Este projeto foi desenvolvido com foco em aprendizado prático, abordando:

- useState

- Atualização de estado baseada no estado anterior

- Manipulação de arrays (map, filter)

- Renderização dinâmica de listas

- Eventos (onClick, onSubmit)

- Lógica condicional

- Organização de componentes


Idéias de futuras melhorias:

- Persistir dados com LocalStorage

- Ranking ou histórico de pontos
